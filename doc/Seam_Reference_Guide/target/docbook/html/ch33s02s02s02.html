33.2.2.2. Configuration and Build changes

resources/META-INF/persistence.xml

  • Because the tech preview version will only work with kodo or openJPA as the JPA provider you must change the provider to : 

    <provider>org.apache.openjpa.persistence.PersistenceProviderImpl</provider>

  • Next you need to change the jta-data-source to what you entered above : 

    <jta-data-source>seam-jee5-ds</jta-data-source>

  • The other properties in the file are hibernate specific and are not used by openJPA but can be left in.
