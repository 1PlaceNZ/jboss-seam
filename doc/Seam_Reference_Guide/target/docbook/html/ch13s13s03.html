13.13.3. Authentication with the Identity Management API

To authenticate using the Identity Management API, it is as simple as not specifying the authenticate-method property for the Identity component. If no authenticate-method is specified, then by default the authentication process (controlled by SeamLoginModule) will attempt to authenticate using IdentityManager's authenticate() method, and no Authenticator component is necessary.
