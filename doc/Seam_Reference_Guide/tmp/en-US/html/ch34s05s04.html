34.5.4.  Building and deploying the seam-gen'd application to Websphere

  • Build your application by calling ant in the base directory of your project (ex. /home/jbalunas/workspace/websphere_example ). The target of the build will be dist/websphere_example.ear .

  • To deploy the application follow the instructions here : Section 34.2.3, “Deploying the application to Websphere” but use references to this project websphere_example instead of jboss-seam-jee5.

  • Checkout the app at: http://localhost:9080/websphere_example/index.html
