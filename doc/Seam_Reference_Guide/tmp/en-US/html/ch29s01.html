29.1. Parameterized Expressions

Standard EL does not allow you to use a method with user defined parameters — of course, JSF listener methods (e.g. a valueChangeListener) take parameters provided by JSF.

JBoss EL removes this restriction. For example: 

<h:commandButton action="#{hotelBooking.bookHotel(hotel)}" value="Book Hotel"/>

@Name("hotelBooking")
public class HotelBooking {
   
   public String bookHotel(Hotel hotel) {
      // Book the hotel
   }
}
