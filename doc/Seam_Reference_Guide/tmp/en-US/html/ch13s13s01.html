13.13.1. Configuration

Configuration of the IdentityManager is extremely simple, requiring only an IdentityStore to be configured in components.xml. The identity management namespace is http://jboss.com/products/seam/security/management and its schema location is http://jboss.com/products/seam/identity-management-2.1.xsd. Here's a simple example showing the configuration of a JPAIdentityStore - for the IdentityManager to use it, it must be named identityStore: 

  <identity-management:jpa-identity-store name="identityStore" account-class="com.acme.UserAccount"/>
