1. Help Contribute

If you find a typographical error in the Seam Reference Guide, or if you have thought of a way to make this manual better, we would love to hear from you! Please submit a report in JIRA: http://jira.jboss.com/jira/browse/JBSEAM against the project Seam and component Documentation.

If you have a suggestion for improving the documentation, try to be as specific as possible when describing it. If you have found an error, please include the section number and some of the surrounding text so we can find it easily.

Note

Be sure to give us your name so you can receive full credit.

Note

This content is taken from https://svn.jboss.org/repos/seam/trunk/doc/ and has yet to be branched.

To access the content directly and make changes yourself: 

svn co https://svn.jboss.org/repos/seam/trunk/doc/ --username yourname
