Get Up And Running Quick
------------------------
# Download JBoss AS 4.0.3
http://www.jboss.com/products/jbossas/downloads
# Install JBoss AS 4.0.3 with the EJB 3.0 profile selected.
# Start JBoss AS 4.0.3 with the bin/run.(sh|bat) script.
# Edit build.properties in your Seam directory and configure the path to your JBoss AS 4.0.3 installation directory.
# In the jboss-seam/examples/booking directory, type ant deploy and check for any error messages.
# Point your browser to http://localhost:8080/seam-booking/ and register an account, book a room...
