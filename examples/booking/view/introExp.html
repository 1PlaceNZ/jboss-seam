What is JBoss Seam

Compared with traditional web development technologies, the JBoss Seam only requires a fraction of the code, needs very little XML configuration files, and eliminates a whole class of bugs. In technical terms, the Seam stack allows us to write the entire application using plain old Java objects (POJOs) and simple XHTML. There is no more boiler plate code and no more trivial multi-layer abstraction / delegation. Seam ties together EJB 3.0 business components with JavaServer Faces (JSF) front-end under a consistent and stateful component model. In this sample, we use Facelets for JSF pages. Facelets allows us to write XHTML page segments and assemble them in a template.

Just like all JBoss software products, this entire software stack is free. The full source code of this hotel booking application is available in the examples/booking directory of the Seam source code distribution. Please read the README file for building instructions. You can learn more about the JBoss Seam framework in the Seam documentation. A 10-minute flash demo on how to build a Seam web application from ground up is also available.
