Welcome to the DVD Store

Welcome to the JBoss Seam DVD Store Demo. The DVD Store is a simple store front application written using the latest JBoss technologies: EJB3, Seam and jBPM.

How to use the DVD Store

There are 5 customer accounts, user1 through user5 (with password "password"). You can login using any of the customer accounts or create your own customer account to browse the DVD store to make purchases. Once an order is submitted, you can check the status of the order from the My Orders screen. When the order is fulfilled by the store manager, the status of the order will be updated.

When a customer places an order, a jBPM order management process. Depending on the version of the process deployed, the store manager (username "manager", password "password") will be presented with different fulfilment options.
