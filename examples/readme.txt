Seam Example Applications
=========================
This directory contains the Seam example applications, which
have been tested in JBoss 4.0.3 rc 3.

registration/   A simple example for the tutorial
booking/        The Seam Booking demo application for EJB 3.0
bpm/            An example using jPBM (work in progress)
noejb/          The Seam Booking demo ported to Hibernate3
dvdstore/       The DVD Store example ported to Seam
portlet/        Based on the booking exemple, this is a portlet working in JBoss Portal
