Yet Affordable.

Just like all our software products, this is free. JBoss is proud to bring you the Seam framework, a whole new way to build web applications in Java.

Creating powerful enterprise applications has never been this easy. Seam combines Enterprise JavaBeans 3.0 components with rich component-based user interfaces built using JavaServer Faces, and lets developers build more robust applications more quickly and with astonishingly little code. Seam helps eliminate a whole class of bugs that plague web development by introducing the concept of declarative application state management.

Get 'Rich' Fast.

Seam is EJB for the web. Now you can write reuseable enterprise components as annotated plain Java objects, and integrate them directly with your presentation tier. You get to decide your architecture. If you don't want to write thousands of lines of code that pass trivial method calls down through various useless layers of tedious abstraction while transforming persistent data between five or six different representations, implementing the same data validation logic in three different ways, you don't have to. Or, if you like that stuff, you can do that, too. Seam accommodates your architecture. And that makes you more productive. You can stop thinking about your container, and start thinking about your application.
